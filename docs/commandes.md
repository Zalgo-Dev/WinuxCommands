
# Commandes disponibles

| Commande | Description                  | Exemple                |
|----------|------------------------------|------------------------|
| `ls`     | Liste les fichiers/dossiers. | `ls`                  |
| `cat`    | Affiche un fichier texte.    | `cat fichier.txt`     |
| `clear`  | Efface la console.           | `clear`               |
